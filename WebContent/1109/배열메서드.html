요소 찾기

item의 위치값을 찾는다 / start 위치 부터 item의 위치값을 찾는다
indexOF(item) / indexOF(item,start)
lastIndexOf(item) / lastIndexOf(item,start)


추가/삭제

push(item) : 배열의 마지막 요소에 item을 추가
pop(): 배열의의 마지막 요소를 제거


sort

nums = [234, 10, 134, 250, 300, 3]
nums.sort() : 자리수별 숫자크기를 판별
오름차순 nums.sort(function(a,b) {a-b})
내림차순 nums.sort(function(b,a) {a-b})


splice-수정(삭제/추가)

splice(index,n) - index 위치에서 n개를 삭제
splice(index, n, a,b,c) - index 위치에서 n개를 삭제하고 a,b,c를 추가
splice(index, 0, a,b,c) - index 위치에서 삭제하지 않고 a,b,c를 추가
