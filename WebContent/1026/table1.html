 	속성 : border = 1
 tr : row, 한줄
 td : 데이터, 한칸
 th : 데이터의 제목
 caption : 테이블 제목
 
 여러개의 table의 스타일을 각각 다르게 지정하기 위해서 id를 부여
 table th : table의 후손 th
 table > tr : table의 자식 tr
