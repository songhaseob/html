Location

- 자동으로 특정페이지로 이동할때 사용 - location.href="페이지이름"
- a 태그에서 이용 함수호출 : javascript:함수명() 공지사항목록
- 이벤트 서버전송(get)- location.href=페이지이름
- 새로운 문서로 대체-location.replace(페이지이름)
- replace()메서드는 현재 페이지를 대체 하기 때문에 뒤로 아이콘이 비활성화 된다
- 뒤로 가기가 실행되지 않는다
- 현재 문서를 다시 리로드
문자열을 배열로 생성하고 랜덤을 발생하여 문자열을 선택한다
선택된 문자열을 result2에 출력하고 1초후에 원래 문서로 돌아온다
